# Project Setup

This project was created with:
- React
- Vite
- Tailwind CSS
- yarn

&nbsp;
## To run this project
&nbsp;

### If you're using yarn:
&nbsp;

This will get all dependencies for you
```bash
  yarn
```

And this will run yout project on localhost
```bash
  yarn dev
```
&nbsp;
### If you're using npm:
&nbsp;

This will get all dependencies for you
```bash
  npm i
```

And this will run yout project on localhost
```bash
  npm run dev
```
