# Project Setup

This project was created with:
- React
- Vite
- Tailwind CSS

## To run this project

-If you're using yarn:

This will get all dependencies for you
```bash
  yarn
```

- And this will run yout project on localhost
```bash
  yarn dev
```
-If you're using npm:

This will get all dependencies for you
```bash
  npm i
```

- And this will run yout project on localhost
```bash
  npm run dev
```
